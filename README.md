This is a Blog Management Web Application built using HTML, CSS, JavaScript on the frontend and Node.js with Express.js on the backend.  
The project demonstrates core CRUD operations (Create, Read, Update, Delete) similar to a Task Management Application, implemented in the form of a blog system.

Users can:
- Create a new blog post  
- View all blog posts  
- Edit existing posts  
- Delete posts  

This project was developed as part of a Full Stack Development Internship Skill Assessment.
